## 1. 功能概述

### 1.1 核心願景
讓用戶能夠與團隊成員、同事或朋友**共享日誌、協作評論、共享決策模板**，建立人機信任與知識共享的社群。

### 1.2 目標市場
- **企業用戶** (HR、團隊領導): 5-10% 市場增長
- **教育工作者** (AI 課程教學): 5-10% 市場增長
- **研究團隊** (AI 倫理、心理學): 3-5% 市場增長
- **預期影響**: +30% 企業客戶、5-10x 高客戶生命週期價值

## 2. 功能需求清單

### 2.1 核心功能 (MVP)

| 功能 | 優先級 | 說明 | 工期 |
|------|--------|------|------|
| **日誌共享** | P0 | 分享個人日誌給特定用戶/群組 | 2 天 |
| **權限控制** | P0 | 讀取、評論、編輯三級權限 | 2 天 |
| **群組管理** | P1 | 建立/管理分享群組 | 2 天 |
| **實時協作** | P1 | WebSocket 即時更新與協作指示器 | 3 天 |
| **評論線程** | P1 | 在條目上添加評論與回覆 | 2 天 |
| **共享模板** | P2 | 保存決策模板供團隊使用 | 1 天 |
| **活動日誌** | P1 | 團隊成員互動記錄 | 1 天 |

### 2.2 UI/UX 流程

#### 共享流程
```
Post → "Share" 按鈕 → Share Modal
  ↓
選擇用戶/群組 → 設定權限 → 確認
  ↓
被邀用戶收到通知 → 接受/拒絕
  ↓
共享日誌出現在用戶的 "Shared with me" 列表
```

#### 協作流程
```
查看共享日誌 → 看到誰在線/編輯
  ↓
添加評論 → 實時更新給所有參與者
  ↓
原作者可接受/拒絕評論建議
  ↓
協作歷史保存在活動日誌
```

## 3. 數據庫設計

### 3.1 新增表結構

#### Table: `shares` (日誌共享記錄)
```python
class Share(Base):
    __tablename__ = "shares"

    id = Column(String(36), primary_key=True)  # UUID
    post_id = Column(String(36), ForeignKey("posts.id"), nullable=False, index=True)
    owner_id = Column(String(36), ForeignKey("users.id"), nullable=False)
    shared_with_id = Column(String(36), nullable=True)  # Specific user, NULL if group
    group_id = Column(String(36), ForeignKey("groups.id"), nullable=True)

    permission = Column(String(20), default="read")  # "read", "comment", "edit"
    status = Column(String(20), default="pending")  # "pending", "accepted", "rejected"

    created_at = Column(DateTime, default=utcnow)
    accepted_at = Column(DateTime, nullable=True)
    expires_at = Column(DateTime, nullable=True)  # Optional expiration

    # Indexes
    Index('ix_shares_post_owner', 'post_id', 'owner_id')
    Index('ix_shares_shared_with', 'shared_with_id')
    Index('ix_shares_group', 'group_id')
```

#### Table: `groups` (協作群組)
```python
class Group(Base):
    __tablename__ = "groups"

    id = Column(String(36), primary_key=True)  # UUID
    name = Column(String(255), nullable=False)
    description = Column(Text, nullable=True)
    creator_id = Column(String(36), ForeignKey("users.id"), nullable=False)

    visibility = Column(String(20), default="private")  # "private", "team", "public"
    icon = Column(String(100), nullable=True)  # emoji or icon name

    created_at = Column(DateTime, default=utcnow)
    updated_at = Column(DateTime, default=utcnow, onupdate=utcnow)

    # Relationships
    members = relationship("User", secondary="group_members", back_populates="groups")
```

#### Table: `group_members` (群組成員)
```python
class GroupMember(Base):
    __tablename__ = "group_members"

    group_id = Column(String(36), ForeignKey("groups.id"), primary_key=True)
    user_id = Column(String(36), ForeignKey("users.id"), primary_key=True)

    role = Column(String(20), default="member")  # "admin", "member"
    joined_at = Column(DateTime, default=utcnow)

    Index('ix_group_members', 'group_id', 'user_id')
```

#### Table: `comments` (協作評論)
```python
class Comment(Base):
    __tablename__ = "comments"

    id = Column(String(36), primary_key=True)  # UUID
    post_id = Column(String(36), ForeignKey("posts.id"), nullable=False, index=True)
    user_id = Column(String(36), ForeignKey("users.id"), nullable=False)

    content = Column(Text, nullable=False)
    is_suggestion = Column(Boolean, default=False)  # "建議編輯" vs "評論"
    status = Column(String(20), default="open")  # "open", "accepted", "rejected", "resolved"

    parent_id = Column(String(36), ForeignKey("comments.id"), nullable=True)  # 回覆關係

    created_at = Column(DateTime, default=utcnow)
    updated_at = Column(DateTime, default=utcnow, onupdate=utcnow)

    Index('ix_comments_post', 'post_id')
    Index('ix_comments_user', 'user_id')
```

#### Table: `activity_logs` (活動日誌)
```python
class ActivityLog(Base):
    __tablename__ = "activity_logs"

    id = Column(String(36), primary_key=True)  # UUID
    group_id = Column(String(36), ForeignKey("groups.id"), nullable=True)
    actor_id = Column(String(36), ForeignKey("users.id"), nullable=False)

    action = Column(String(50), nullable=False)  # "share_post", "comment", "accept_share", "create_group"
    target_type = Column(String(50), nullable=False)  # "post", "comment", "user"
    target_id = Column(String(36), nullable=False)

    metadata = Column(JSON, nullable=True)  # Additional context

    created_at = Column(DateTime, default=utcnow, index=True)

    Index('ix_activity_group', 'group_id', 'created_at')
```

## 4. API 端點設計

### 4.1 共享相關

#### POST /api/v1/posts/{post_id}/share
**創建共享**
```json
{
  "shared_with_ids": ["user_1", "user_2"],
  "group_ids": ["group_1"],
  "permission": "comment"
}
```

#### GET /api/v1/posts/shared-with-me
**獲取共享給我的日誌**
```json
{
  "shares": [
    {
      "id": "share_id",
      "post": {...},
      "owner": {...},
      "permission": "comment",
      "status": "accepted",
      "shared_at": "2026-04-01T10:00:00Z"
    }
  ],
  "total": 5
}
```

#### DELETE /api/v1/shares/{share_id}
**撤銷共享**

## 6. 實時協作架構

### 6.1 WebSocket 實現
```python
# FastAPI + python-socketio
from fastapi import FastAPI
from python_socketio import AsyncServer

# 連接管理
sio = AsyncServer(async_mode='asgi', cors_allowed_origins='*')
app = FastAPI()
app = socketio.ASGIApp(sio, app)

@sio.event
async def connect(sid, environ):
    user_id = authenticate_socket(environ)
    active_users[sid] = user_id

@sio.on('join_post')
async def join_post(sid, data):
    post_id = data['post_id']
    await sio.enter_room(sid, f"post_{post_id}")
    await sio.emit('user_joined',
        {'user_id': user_id, 'user_name': user.name},
        room=f"post_{post_id}")
```

### 6.2 衝突解決
- 使用樂觀鎖 (version field) 或操作轉換 (OT)
- 評論是追加, 不存在衝突
- 建議編輯使用 diff 進行版本控制
