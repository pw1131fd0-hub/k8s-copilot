### 4. Key Features

#### Language Detection
```javascript
// Automatically detects browser language on first visit
const getBrowserLanguage = () => {
  const browserLang = navigator.language;
  if (browserLang.startsWith('zh-TW')) return 'zh-TW';
  if (browserLang.startsWith('zh')) return 'zh';
  if (browserLang.startsWith('ja')) return 'ja';
  return 'en'; // default
};
```

#### Language Persistence
- User's language preference saved to `localStorage`
- Remembered across browser sessions
- Can be cleared through settings (future feature)

#### Fallback Mechanism
```javascript
// If a translation is missing, falls back to English
fallbackLng: 'en'
```

#### Instant Language Switching
- No network calls required
- All translations loaded upfront
- Instant UI update when user switches languages

## Developer Notes

### Adding New Translations
1. Add key to English file: `en.json`
2. Add same key to other language files: `zh.json`, `zh-TW.json`, `ja.json`
3. Use in component: `const { t } = useTranslation(); t('key.name')`

### Example Component
```jsx
import { useTranslation } from 'react-i18next';

export function MyComponent() {
  const { t } = useTranslation();

  return (
    <div>
      <h1>{t('common.appName')}</h1>
      <p>{t('home.subtitle')}</p>
      <button>{t('common.save')}</button>
    </div>
  );
}
```

### Changing Language Programmatically
```jsx
const { i18n } = useTranslation();
await i18n.changeLanguage('ja'); // Switch to Japanese
```
